DRIP Score™ Methodology
How we calculate protocol risk
ComponentWeightFactors ConsideredScore Range
Code Safety30%Number of audits, age of protocol, bug bounty program, code complexity0-100
Drawdown Risk25%Historical max TVL drawdown, speed of recovery, correlation to ETH price0-100
Volatility20%TVL standard deviation, 7-day/30-day change acceleration, inflow/outflow spikes0-100
Concentration15%Single-chain dependency, top-wallet concentration, collateral type diversity0-100
Liquidity Risk10%Exit liquidity depth, slippage at scale, oracle dependencies, DEX liquidity0-100
DRIP Score = (Code×0.30) + (Drawdown×0.25) + (Volatility×0.20) + (Concentration×0.15) + (Liquidity×0.10) - Scores below 22 are HEALTHY, 22-35 MODERATE, 35-50 ELEVATED, above 50 HIGH RISK.
